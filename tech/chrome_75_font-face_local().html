@font-face local() does not work properly for multi weight typeface


@font-face {
	font-family: r;
	font-weight: 400;
	src: local("Times New Roman");
}


p style="font-family:r; font-weight:400;"

// It works fine for regular font.



style="font-family:'Times New Roman'; font-weight:700;"

// This is the bold weight I want.



@font-face {
	font-family: b;
	font-weight: 700;
	src: local("Times New Roman Bold");
}

style="font-family:b;font-weight:700;"

// On this condition, chrome will not detect the local bold variant. Meanwhile, MS browsers work properly:

MS Edge:

IE11:



style="font-family:'Times New Roman'; font-weight:700;"

// This is the bold weight I want.
