Web Safe Fonts 指的是在各个操作系统广泛预装，方便网页直接调用的字体。

这个页面将会尝试调用常见的 Web 安全字体，测试你的操作系统和浏览器对其的支持情况。

或者你也可以把本页面当作一个字体挑选器来使用。

初步的候选字体列表来自 cssfontstack.com。最终的列表经我在 Apple 与 微软 的官网逐个确认手工筛选得到。

2019/06/07

I 西文无衬线体

Arial | [Apple] Helvetica

Helvetica是一种非常著名、非常经典的的无衬线字体。你可以在数百家知名企业的 logo 与宣传资料中见到它。
Helvetica 也是旧版 iOS 与 macOS 的默认字体，目前被 San Francisco 代替。
可惜的是，微软未曾购买过 Helvetica 字体的版权，而是采购了与 Helvetica 非常相似的Arial。由于这样的历史原因，目前 Arial 的覆盖度要比 Helvetica 好得多。

Tahoma

90年代 Matthew Carter 为微软设计的网页字体，现在已经成为事实上的跨平台网页设计标准字体之一。
Tahoma 是 Windows 2000、XP、2003的默认字体，自 Vista 之后被 Segoe UI 代替。

Verdana

90年代 Matthew Carter 为微软设计的网页字体，现在已经成为事实上的网页设计标准字体之一。Verdana 也是目前宜家家居指南所用的字体。

[Android] [Web] Roboto

Android 的默认无衬线字体。Roboto 字体整个系列都采用 Apache 协议开源，在非 Android 平台上很容易调用到 Web Font。

Trebuchet MS

90年代诞生的专为网页排版设计的微软字体，Trebuchet MS 的字形要比同一时代的另几套字体更加有个性。Windows xp 默认主题有一些UI使用的字体就是是 Trebuchet。

[Win] Comic Sans MS

微软提供的一种漫画风格的艺术字体。Windows 平台的预装字体，同时也是 iOS 的可下载字体。

Arial Black
Arial 的粗体，非常醒目，覆盖面非常广。

[Win][mac] Impact

另一种非常醒目的粗体。

II 西文衬线体

Times New Roman

可能是最常见且广为人知的衬线字体之一，在字体设计上属于过渡型衬线体，对后来的字型产生了很深远的影响。 《泰晤士报》首次采用了 Times New Roman 后，这个字型很快地博得了大众的青睐，获得了极大的成功。尽管泰晤士报已不再使用Times New Roman，这个字形已成为经典。

Georgia

Matthew Carter 为微软设计的几套网页字体中，Georgia 是作为与 Verdana 相对的衬线体而存在的。
从20年前的低精度显示器直到目前的视网膜屏，Georgia 一直是一款适合屏幕阅读的字体。Georgia 的一个很有辨识度的特色是默认的 old style 的数字：1234567890。也可以通过 CSS 指定为等高风格的数字 1234567890

[Win] Platino Linotype | [Apple] Platino[Ubuntu] URW Palladio L

一种有着怀旧意大利风格的旧衬线体。Palatino 也是一个默认 old style 数字的字体：1234567890 。当然，也可以通过 CSS 指定为等高风格的数字 1234567890

[Office][mac] Garamond

Garamond 是一类源于16世纪法国印刷版的旧衬线体的统称。目前存在非常多的数字化版本，例如微软 Office 和 macOS 都提供了自己的 Garamond。
作为一种古代字体，Garamond 在低分辨率屏幕的显示效果是不如 Georgia 与 Palatino的。好在本博客自始至终就不是面向低分辨率屏幕、IE浏览器这类过时终端设计的，因而可以任性地选择 OFL 协议开源的 EB Garamond 的一个 Variable Font 版本作为首选的西文 Web Font。仅靠一个字体文件即可提供 400-800 的字重、以及丰富的连字liga、花体Swash特性，用有限的传输大小，覆盖了从标题字到正文字的应用。

[Apple][Office]Rockwell

一种长青的粗衬线体，比一般的衬线体更加富有活力。 macOS 与 iOS 预装，Office 附带。

III 西文等宽体

Courier New
这是目前仅有的一个在各个平台均能很好显示的等宽字体。

Ubuntu Mono
Ubuntu预装的等宽字体。

Source Code Pro
Adobe出品的开源等宽字体，在程序员的电脑上有一定的覆盖率。有web font

IV 简体中文黑体

黑体一般不需要特别声明具体字体。

苹果生态默认的苹方和 Android 默认的 Noto Sans 都是很好的选择。

唯一需要注意的是，某些版本的 Windows 可能会把 sans-serif 回落到中易宋体，因此最好在通用类前声明一个微软雅黑。

[Win] 微软雅黑 / 微软雅黑 Light

方正为微软打造的 Windows 平台的简体中文默认字体，预装于 Vista 及之后的所有版本 Windows 和 Office 中。
微软雅黑提供 Regular 和 Bold 字重。Windows 8.1 新增的 微软雅黑 Light 汉字部分来自相同字重的方正兰亭黑，与微软雅黑同源，但并不是同一款字体。
微软雅黑 和 方正兰亭黑 的中宫肥胖，设计之初即考虑了非视网膜屏幕上的小字显示效果。微软还在历年的系统升级中对这套字体进行过多次微调，目前微软雅黑的版本号已经高达 6.25。十分适合作为网页在 Windows 平台的首选正文字体。

[mac] 冬青黑体

OS X 10.6 至 macOS 10.10 的简中默认字体，提供W3、W6两种字重。目前被 苹方-简 代替，但仍随macOS预装，iOS 设备可通过 Font Book下载。
冬青黑体是日文冬青体的简体中文版本。日系风格大陆简体中文字形规范相结合，吸引了一大批拥趸。如果能够在 Windows 浏览器上调用到用户自行安装的冬青黑体，也不奇怪。

[Apple] 苹方-简

Apple 设备的简体中文默认字体，去掉了 黑体-简 饱受诟病的喇叭口，为屏幕优化，字重由两种增加为六种。
是目前 Apple 设备所预装的中文字体之中功能性最强的。

[mac] 黑体-简

由华文黑体、华文细黑修改而来，提供两种字重。是 iOS9 之前 iOS 设备的简中默认字体。
目前 黑体-简 已被 苹方-简 代替。macOS 仍然预装，iOS 设备需下载。

[Office] 华文细黑

一种随 Office 附带的字体，Apple 设备可通过 Font Book下载。

[Android][Web] 思源黑体

Android 设备预装的泛中日韩默认字体，提供7种字重。不过由于Android 系统碎片化的缘故，许多设备仅提供一个 Regular 字重。
有一些中文用户也会在桌面上安装思源黑体，Android 和桌面的体验。不过有一点需要注意，思源黑体的西文部分是 Adobe Source Sans Pro ，而用 Roboto 来搭配思源黑体才是原汁原味的 Android 风格。

[Android] Droid Sans / [Linux] 文泉驿微米黑

更加古老的 Android 的默认字体，一些中文 Linux 系统上也会预装一个叫做文泉驿微米黑的字体，是 Droid Sans 的调整版本。

V 简体中文宋体

随着视网膜屏的普及，屏幕与印刷的精度差距有所减小。宋体作为传统的中文正文字体，也开始慢慢回潮。

然而，2019年的各大操作系统，要么压根就没有内置宋体，即使有也是内置的质量过时的老旧字体。因此网页上的宋体使用 Web Font 几乎是不可避免的。

从易得性和功能性上来说，思源宋体可以说是目前宋体 Web Font 的最佳选择。声明 font-family 时可以在思源宋体后面同时声明一些常见的宋体，作为浏览器下载字体时的fallback。

[Win] 中易宋体 / 新宋体

饱受诟病的 Windows 平台默认简中衬线体，且为 Windows XP 之前的简中系统默认字体。这个远古字体的屏幕显示效果极差。

[mac] [Office] 华文宋体

macOS 预装的一种宋体，Windows 平台上会随 Office 中文版附带。同中易宋体一样，华文宋体也没有为屏幕显示优化。仅提供单一字重。细节上，华文宋体的西文要比中易宋体的耐看一些。
事实上，对于网页排版来说，华文宋体所提供的字重轻了些。为了避免过于频繁地使用仿粗体，应当考虑与同为 Office 附带的 华文中宋 搭配使用。

[mac] 宋体-简

华文为 macOS 10.12 定制的宋体，macOS预装， iOS 可下载。提供 Light、Regular、Bold、Black 四种字重。推荐作为 macOS 上显示的首选宋体。

[Web] 思源宋体 / Source Han Serif / Noto Serif CJK

一种新兴的 SIL 协议开源字体。覆盖 Unicode 字符集的 CJK 字形和七种字重带来非常全面的功能性。
目前还没有任何操作系统预装思源宋体，但我们可以通过在 Adobe Fonts、Google Fonts 等平台调用 Web Font 的方式来将思源宋体引入网页。

VI 简体中文仿宋

仿宋和楷体常常用来搭配西文的斜体使用

可惜的是 chrome 还不支持禁用字体合成，导致中西文混排时非常容易出现仿宋跟着西文一齐倾斜的情况

一种折中的方法是 用@font face 把仿宋/楷体的中文字体声明为斜体，浏览器便不会合成伪斜体了

虽然 CSS 允许把 fangsong 设置为 font-family 的通用字族，然而实际上，mac / iOS / Android 都没有预装可供回落的仿宋字体——我为这段话声明了仿宋，但你所看到的很可能仍然是系统默认的黑体。

[Win] 中易仿宋

仿宋 是 Windows Vista 及之后 Windows 所预装的仿宋字体，而仿宋_GB2312 是 Windows XP 及更老版本 Windows 所预装的仿宋字体，仅包含6763个汉字，很容易出现生僻字无法显示的情况。

[Office] 华文仿宋

Office 中文版附带，macOS 和 iOS 可下载。华文仿宋的西文字形要比中易仿宋好看些。

VII 简体中文楷体

楷体甚至都不是 CSS 的通用字族，目前只有 Windows 预装，不过 macOS 也包含了一个多字重的楷体作为可下载选项。

[Win] 中易楷体

楷体 是 Windows Vista 及之后的 Windows 中预装，而楷体_GB2312 在 Windows XP 及更老版本的 Windows 中预装。中易楷体的西文很诡异地选用了无衬线体。

[Office] 华文楷体

Office 中文版附带，macOS 和 iOS 可下载。华文楷体的西文字形还是要比中易楷体好看些。

[Apple] 楷体-简

华文为最新的 macOS 10.13 优化的楷体，但并不随系统预装，需下载。提供了 Regular，Bold，Black 三种字重。
